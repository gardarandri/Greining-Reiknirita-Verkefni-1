# Greining Reiknirita: Verkefni 1

Hér má finna kóðan fyrir verkefni 1.

* gas\_qsort.c sér um að kalla á aqsort fyrir mismunandi stæriðr af inntökum og vendistakavali 
* gas\_qsort.h inniheldur útfærslu á Quicksort og mismunandi vendistakavölum
* aqsort.h er skrá sem hefur fallið aqsort sem notað er í gas\_qsort.c

Skipunin

> make info

gefur grunn upplýsingar um mismunandi tegundir af vendistakavali. Skipunin

> make data

spítir út gögnum sem má nota til að búa til myndir af mótdæmisfylkjunum fyrir mismunandi gerið vendistakavals
